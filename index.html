Project Overview

Design and implement an edge-first computer vision system for leak detection and environmental monitoring in industrial sites. The system runs on NVIDIA Jetson Nano hardware, enabling real-time, offline-capable detection directly at the edge without reliance on constant network connectivity.

Project Timeline & Activities

October 2

Project Requirements & Strategy

Defined project requirements and discussed strategies to approach the problem. The team decided to begin with tank-based leak detection as the primary focus. Explored LabelImg in Python for the image labeling process.

October 13

Data Collaboration with SitePro

Collaborated with SitePro to gather relevant data, including leak and no-leak images, along with corresponding dates, camera IDs, and site IDs for analysis.

October 14

Image Extraction Pipeline

Built the image extraction pipeline, successfully downloading and organizing images into ZIP files categorized by weather conditions, time of day, and camera angles.

October 16

Dataset Labeling

Began labeling images into leak and no-leak categories to prepare the dataset for model training.

October 17

UML Design Phase

Initiated the UML design phase, developing key diagrams to represent the system architecture and workflows.

October 18

YOLO Model Training

Started training the YOLO-based model using the labeled dataset to identify potential leaks.

UI Prototype Design

Designed the low-fidelity user interface prototype in Figma to visualize the planned system layout.

October 20

Technical Documentation

Began drafting the technical report and presentation, summarizing the progress, design, and model performance achieved during this stage.
